# Electron

A new-gen ecommerce webapp

# How to run locally

1. Clone the repository locally

`git clone https://github.com/Condition00/electron.git`

## Running the frontend

1. Change into the `client` directory using `cd client` and run

`npm i`

After the command is completed

2. Run the server locally by running

`npm run dev`

!!! Make sure you're inside the client directory when you run these commands
