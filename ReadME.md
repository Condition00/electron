# Electron

A new-gen ecommerce webapp

# How to run locally

1. Clone the repository locally

`git clone https://github.com/Condition00/electron.git`

## Running the frontend

1. Install frontend dependencies:

`cd client && npm i`

2. Start frontend (from project root):

`npm run dev`

or directly from client:

`cd client && npm run dev`

## Running the backend

1. Install backend dependencies:

`cd server && npm i`

2. Start backend (from project root):

`npm run dev:server`
